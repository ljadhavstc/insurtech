Quick Start

Installation

npm install
# or
yarn install

Run the App

# Start Metro bundler
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android

Generate Tokens from Figma

# Paste Figma variables JSON and run:
npm run generate-tokens

# Or with custom input:
cat figma-variables.json | npm run generate-tokens
