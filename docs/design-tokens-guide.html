üí™ Font Weights

Use font weight utilities to adjust text emphasis.

Weight Value Usage Class
Thin 100 Very light text font-thin
Light 300 Light text font-light
Regular 400 Normal text (default) font-regular or font-normal
Medium 500 Medium emphasis font-medium
Semibold 600 Semi-bold text font-semibold
Bold 700 Bold text font-bold
Extrabold 800 Extra bold text font-extrabold
Black 900 Heaviest text font-black

Font Weight Examples

Visual Examples
Thin (100) - Very light text
Light (300) - Light text
Regular (400) - Normal text
Medium (500) - Medium emphasis
Semibold (600) - Semi-bold text
Bold (700) - Bold text
Extrabold (800) - Extra bold text
Black (900) - Heaviest text

Usage Examples

// Using font weight classes
<Text className="text-theme-text-primary font-bold">
  Bold text
</Text>

<Text className="text-theme-text-primary font-semibold">
  Semibold text
</Text>

// Combined with typography
<Text className="text-body font-medium text-theme-text-primary">
  Medium weight body text
</Text>

// Using composite text styles (includes weight)
<Text className="text-text-text-bold text-theme-text-primary">
  Already includes bold weight
</Text>
