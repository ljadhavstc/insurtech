Usage Guide

How to use the InsurTech App scaffold

Getting Started

1. Install Dependencies

npm install
# or
yarn install

2. Generate Design Tokens from Figma

Export your Figma variables as JSON and run:

npm run generate-tokens

Or with a file:

cat example-figma-variables.json | npm run generate-tokens

3. Start Development

# Start Metro bundler
npm start

# In another terminal, run on device:
npm run ios
# or
npm run android

Form Components

import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';
import { FormField } from '@/components/form';

const schema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
});

const { control, handleSubmit } = useForm({
  resolver: zodResolver(schema),
});

<FormField
  control={control}
  name="email"
  label="Email"
  placeholder="Enter email"
/>

Scaling Utilities

Use scaling functions for responsive design:

import { s, vs, ms, fs } from '@/utils/scale';

// Scale width (based on screen width)
const width = s(100);

// Scale height (based on screen height)
const height = vs(50);

// Scale padding/margin (moderate scaling)
const padding = ms(16);

// Scale font size
const fontSize = fs(16);
Tip: Use ms() for padding and margins, fs() for font sizes, and s() or vs() for specific width/height needs.

Styling Approach

Combining Tailwind Classes and Scaled Props

// Use className for colors, layout, and Tailwind utilities
// Use scaled props (p, m, size) for Figma-specified dimensions

<Box
  p={16}                    // Scaled padding from Figma
  className="bg-white rounded-lg flex-row items-center"
>
  <Text
    size={16}               // Scaled font size from Figma
    className="text-primary font-semibold"
  >
    Text
  </Text>
</Box>
Best Practice: Use Tailwind classes for colors, layout (flex, grid), and utilities. Use scaled props for dimensions specified in Figma designs.

Authentication Flow

Login

import api from '@/services/api';
import { authStore } from '@/stores/authStore';

const response = await api.post('/auth/login', {
  email: 'user@example.com',
  password: 'password123',
});

const { user, token, refreshToken } = response.data;
authStore.getState().login(user, token, refreshToken);

Check Auth Status

import { authStore } from '@/stores/authStore';

const isAuthenticated = authStore((state) => state.isAuthenticated);
const user = authStore((state) => state.user);

Logout

authStore.getState().logout();

Internationalization

import { useTranslation } from 'react-i18next';

const { t, i18n } = useTranslation();

// Translate
const title = t('auth.login.title');

// Change language
i18n.changeLanguage('es');

Toast Notifications

import { useToast } from '@/components/Toast';

const { showToast } = useToast();

showToast({

  message: 'Saved successfully!',
  duration: 3000, // optional
});

Mock Server

The app includes a mock server for development. Default credentials:

  • Email: user@example.com
  • Password: password123

Testing

# Run tests
npm test

# Run with coverage
npm test -- --coverage
Note: Remember to replace the mock server with actual API endpoints in production. Update src/services/api.ts with your production base URL.
