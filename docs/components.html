Components

Reusable UI components and primitives

Primitive Components

Text

NativeWind-styled Text wrapper with size scaling and className support.

import { Text } from '@/components/primitives/Text';

<Text size={16} className="text-primary font-semibold">
  Hello World
</Text>

<Text variant="h1" className="text-text-primary">
  Heading
</Text>
PropTypeDescription
sizenumberFont size in Figma px (scaled automatically)
variantTypographyVariantTypography variant (h1, h2, body, etc.)
classNamestringTailwind classes
allowFontScalingbooleanDefault: true

Box

View wrapper with scaled padding/margin props and className support.

import { Box } from '@/components/primitives/Box';

<Box p={16} m={8} className="bg-white rounded-lg">
  Content
</Box>
PropTypeDescription
p, px, py, pt, pb, pl, prnumberPadding in Figma px (scaled)
m, mx, my, mt, mb, ml, mrnumberMargin in Figma px (scaled)
classNamestringTailwind classes

Button

Button component with variants, loading state, and scaled dimensions.

import { Button } from '@/components/primitives/Button';

<Button variant="primary" onPress={handlePress} loading={isLoading}>
  Click Me
</Button>
PropTypeDescription
variantButtonVariantprimary | secondary | ghost | outline | danger
heightnumberButton height in Figma px (default: 48)
radiusnumberBorder radius in Figma px (default: 8)
loadingbooleanShow loading spinner
disabledbooleanDisable button
fullWidthbooleanFull width button

Input

TextInput with error state, label, and icons.

import { Input } from '@/components/primitives/Input';

<Input
  label="Email"
  placeholder="Enter email"
  value={value}
  onChangeText={onChangeText}
  error={error}
/>
PropTypeDescription
labelstringInput label
errorstringError message
helperTextstringHelper text
leftIconReactNodeLeft icon component
rightIconReactNodeRight icon component
heightnumberInput height in Figma px (default: 48)

IconButton

Button that displays an icon with different sizes and variants.

import { IconButton } from '@/components/primitives/IconButton';

<IconButton
  icon={<Icon name="heart" />}
  onPress={handlePress}
  variant="ghost"
  size="md"
/>

Spacer

Simple component for adding vertical or horizontal spacing.

import { Spacer } from '@/components/primitives/Spacer';

<Spacer height={16} />
<Spacer width={8} />

UI Components

Modal

Reusable modal component with backdrop.

Alert component for displaying messages.

Toast notification system with provider.

const { showToast } = useToast();
showToast({ type: 'success', message: 'Saved!' });
