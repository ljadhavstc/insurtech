Utils

Utility functions and helpers

Scaling Utilities

Location: src/utils/scale.ts

Functions for responsive scaling based on device dimensions.

s(size: number)

Scale a size based on screen width. Use for: width, horizontal sizes.

import { s } from '@/utils/scale';

const width = s(100); // 100px on base design, scaled for device
ParameterTypeDescription
sizenumberSize in pixels from design (Figma)

Returns: Scaled size for current device (rounded)

vs(size: number)

Scale a size based on screen height. Use for: height, vertical sizes (margins, paddings).

import { vs } from '@/utils/scale';

const height = vs(50); // 50px on base design, scaled vertically

ms(size: number)

Scale a size moderately (average of width and height scaling). Use for: padding, margin, border radius.

import { ms } from '@/utils/scale';

const padding = ms(16); // 16px on base design, scaled moderately

fs(size: number)

Scale font size based on screen width with pixel ratio consideration. Use for: fontSize.

import { fs } from '@/utils/scale';

const fontSize = fs(16); // 16px font, scaled for readability

getScreenDimensions()

Get current screen dimensions.

import { getScreenDimensions } from '@/utils/scale';

const { width, height } = getScreenDimensions();

isTablet()

Check if device is a tablet.

import { isTablet } from '@/utils/scale';

if (isTablet()) {
  // Tablet-specific logic
}

isSmallScreen()

Check if device has a small screen.

import { isSmallScreen } from '@/utils/scale';

if (isSmallScreen()) {
  // Small screen-specific logic
}

Base Dimensions

The scaling functions use iPhone 11 Pro dimensions as the base:

  • Base Width: 375px
  • Base Height: 812px

All scaling is relative to these base dimensions to maintain consistent visual appearance across devices.
