RegisterScreen

Path: src/features/auth/screens/RegisterScreen.tsx

User registration screen with name, email, password, and confirm password.

  • Name, email, password, and confirm password fields
  • Password matching validation
  • Form validation with react-hook-form + zod
  • Sign in navigation link
  • Loading state during registration

ForgotPasswordScreen

Path: src/features/auth/screens/ForgotPasswordScreen.tsx

Screen for requesting password reset OTP.

  • Email input field
  • Sends OTP to user's email
  • Navigates to OTP verification on success
  • Back to login link

OTPVerificationScreen

Path: src/features/auth/screens/OTPVerificationScreen.tsx

Screen for verifying OTP code sent to user's email.

  • 6-digit OTP input field
  • OTP validation
  • Resend OTP functionality
  • Navigates to reset password on success

ResetPasswordScreen

Path: src/features/auth/screens/ResetPasswordScreen.tsx

Screen for resetting password after OTP verification.

  • New password and confirm password fields
  • Password matching validation
  • Navigates to login on success
